# Telecom-Customer-Churn-Analysis-Prediction

# 🧩 Project Overview

Customer churn is one of the biggest threats to revenue growth in subscription-based businesses.  
In this project, I developed a complete end-to-end churn analysis solution for a telecom company using Excel, SQL Server, Power BI, and Machine Learning.

The project focused on:
- Understanding why customers leave
- Identifying high-risk customer segments
- Visualizing churn behavior
- Predicting future churners using machine learning

---

# ⚙️ Project Workflow

## 🔹 Data Engineering & ETL
Using SQL Server, I:
- Imported telecom customer datasets into staging tables
- Cleaned and transformed raw data
- Handled null values and inconsistent records
- Built production-ready tables and SQL views for reporting

---

## 🔹 Business Intelligence Dashboard
Built an interactive Power BI dashboard to analyze churn across:

### Customer Demographics
- Gender
- Age Groups
- Marital Status

### Account & Revenue Insights
- Contract Types
- Payment Methods
- Monthly Charges

### Geographic Analysis
- State-level churn distribution

### Service Analysis
- Internet Services
- Streaming Services
